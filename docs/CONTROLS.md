# Controls and Mechanics

## Direction Notation

The motion system uses fighting-game numpad notation:

```text
7 8 9
4 5 6
1 2 3
```

`6` is forward, `4` is back, and directions are interpreted relative to the fighter’s facing.

## Common Motions

- Quarter-circle forward: `236 + P`
- Quarter-circle back: `214 + K`
- Dragon punch: `623 + P`
- Half-circle back: `63214 + P`
- Double quarter-circle forward: `236236 + P`
- Charge back-forward: hold back, then forward + button
- 360: full-circle style command for Titan

## Universal Systems

- **MAX Mode**: `LP + LK`
- **Guard Cancel Roll**: `LP + LK` while blocking
- **Blowback/Shatter Strike**: `HP + HK`
- **Burst**: `LP + LK + HP + HK`
- **Pause**: `Esc`
- **Input Display**: `Tab`

## Training Hotkeys

- `1`: Dummy stand
- `2`: Dummy crouch
- `3`: Dummy jump
- `4`: Dummy block all
- `5`: Dummy random block
- `6`: Dummy wake-up DP
- `7`: Record/playback toggle
- `8`: Normal meter
- `9`: Auto-refill meter
- `0`: Infinite meter
- `R`: Reset positions
- `F1`: Frame data overlay
- `F2`: Hitbox viewer
